You are given a 3D array of integers Boxes where Boxes[i] = [wi, hi, di]
represents the width, height and depth of a box. One box can fit into another if and
only if both the width, height and depth of one box is greater than the width, height and depth
of the other box.
Return the maximum number of boxes that you can put one inside the other.
Note: You can rotate the boxes.

Constraints:

  • 1 <= boxes.length <= 50
  • 1 <= boxes[i].length <= 3
Read initial text from the file input.txt. Write result text to output.txt.

Example

input.txt 
5 4 2 6 4 5 6 7 4 2 3 5
output.txt 
3

Example

input.txt 
1 1 1 2 2 2 3 3 3
output.txt 
3
